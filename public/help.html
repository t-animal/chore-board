Chores - Help

How do I add stuff?

Go to google calendar and add a (recurring) event to the calendar you've chosen to use for your chores.

How do I remove stuff?

If you want to mark a chore as done click on it. You can select whether to hide the item immediately or only after it's due in the sidebar.

If you really want to remove an item for good, use google calendar.

How do I color stuff?

If you choose a color in the google calendar's event, the chore will be displayed using this color.

Why does this website modify the event?

This allows you to see whether a chore is done using other calendar apps, too. Also, you can mark a chore as done by modifying the event's description to start it with "Done" or add the Checkmark character to the title. Either is fine.
